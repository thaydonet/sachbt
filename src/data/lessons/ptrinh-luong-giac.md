## Phương trình lượng giác cơ bản

### $\sin x = m$

Điều kiện: $|m| \leq 1$

$$
x = \arcsin m + k2\pi \quad\text{hoặc}\quad x = \pi - \arcsin m + k2\pi \quad (k \in \mathbb{Z})
$$

Đặc biệt:

- $\sin x = 0 \iff x = k\pi$
- $\sin x = 1 \iff x = \frac{\pi}{2} + k2\pi$
- $\sin x = -1 \iff x = -\frac{\pi}{2} + k2\pi$

### $\cos x = m$

Điều kiện: $|m| \leq 1$

$$
x = \pm \arccos m + k2\pi \quad (k \in \mathbb{Z})
$$

Đặc biệt:

- $\cos x = 0 \iff x = \frac{\pi}{2} + k\pi$
- $\cos x = 1 \iff x = k2\pi$
- $\cos x = -1 \iff x = \pi + k2\pi$

### $\tan x = m$

$$
x = \arctan m + k\pi \quad (k \in \mathbb{Z})
$$

### $\cot x = m$

$$
x = \text{arccot } m + k\pi \quad (k \in \mathbb{Z})
$$

## Phương trình bậc hai với một hàm lượng giác

Dạng: $a \cdot \sin^2 x + b \cdot \sin x + c = 0$ (tương tự với $\cos$, $\tan$, $\cot$)

Cách giải: đặt $t = \sin x$ ($|t| \leq 1$).
