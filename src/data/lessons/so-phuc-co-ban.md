## Số phức

Số phức có dạng $z = a + bi$, trong đó $a, b \in \mathbb{R}$, $i^2 = -1$.

- $a$: phần thực
- $b$: phần ảo
- $i$: đơn vị ảo

### Biểu diễn hình học

Số phức $z = a + bi$ được biểu diễn bởi điểm $M(a; b)$ trên mặt phẳng $Oxy$.

### Môđun

$$
|z| = \sqrt{a^2 + b^2}
$$

Tính chất:

- $|z| \geq 0$, $|z| = 0 \iff z = 0$
- $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$
- $\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}$

### Số phức liên hợp

$\bar{z} = a - bi$

Tính chất:

- $z = \bar{z} \iff z \in \mathbb{R}$
- $\overline{z_1 \pm z_2} = \bar{z}_1 \pm \bar{z}_2$
- $\overline{z_1 \cdot z_2} = \bar{z}_1 \cdot \bar{z}_2$
- $z \cdot \bar{z} = |z|^2 = a^2 + b^2$

### Các phép toán

- $(a + bi) + (c + di) = (a + c) + (b + d)i$
- $(a + bi) - (c + di) = (a - c) + (b - d)i$
- $(a + bi)(c + di) = (ac - bd) + (ad + bc)i$
- $\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2}$
